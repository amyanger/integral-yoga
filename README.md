# Sri Aurobindo & The Mother - Integral Yoga

A web experience exploring the profound teachings and spiritual vision of Sri Aurobindo and The Mother, pioneers of Integral Yoga and the transformation of human consciousness.

## About

This site presents the core teachings of Integral Yoga, including:
- Sri Aurobindo's vision of spiritual evolution
- The Mother's practical guidance
- The concept of the Divine Life
- Transformation of human consciousness

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
